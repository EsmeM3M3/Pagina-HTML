Estructura

(HyperText Markup Language) es el lenguaje estándar para crear páginas web. Aquí tienes una explicación básica de su estructura:

Declaración del tipo de documento: Al principio de un documento HTML, se incluye `<!DOCTYPE html>` para indicar que se está utilizando HTML5.
Elemento `<html>`: Este es el contenedor principal de todo el contenido HTML.
Elemento `<head>`: Contiene metadatos sobre el documento, como el título (`<title>`), enlaces a hojas de estilo, y metaetiquetas.
Elemento `<body>`: Contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, etc.

EJEMPLO

CODIGO DE HTML

<!DOCTYPE html>
<html>
<head>
<title>Mi Página Web</title>
</head>
<body>
<h1>¡Hola, mundo!</h1>
<p>Este es un párrafo de ejemplo.</p>
</body>
</html>
